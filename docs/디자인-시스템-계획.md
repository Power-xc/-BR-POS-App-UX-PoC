# 던킨 매장 관리 앱 — 디자인 시스템 계획

> 작성일: 2026-04-02
> 레퍼런스: Flighty · Skip · GCOO
> 방향: 프리미엄 B2B 운영툴 (콘텐츠앱 X, 대시보드 O)

## 1. 디자인 방향 결정

### 레퍼런스 종합

| 레퍼런스 | 추출 요소 | 적용 위치 |
|---------|---------|---------|
| **Flighty** | 다크 히어로 + 화이트 패널 대비, 바텀시트, 대형 수치 타이포 | 모바일 헤더, BottomSheet, 재고·발주 수치 |
| **Skip** | Full-pill CTA 버튼, 컬러 인포 배너, 구분선 리스트 | 승인·결제 버튼, 특이사항·마감 배너 |
| **GCOO** | 타이머 카운트다운 배너, 트랜잭션 리스트, 날짜 섹션 헤더 | 마감 임박 배너, 리뷰 목록 |

### 버린 것
- 네이버: 콘텐츠 포털 스타일 → B2B 운영툴에 부적합
- Flighty 우주 배경: B2C 감성 과잉
- Skip 오렌지 CTA: 브랜드 컬러 충돌

## 2. 브랜드 컬러 정책

**결론: 블랙 primary 유지 + 던킨 오렌지 5% 포인트**

```
이유:
오렌지 heavy = 음식 배달앱 느낌
점주가 하루 8시간 보는 대시보드 → 눈 피로도 감소 우선
Toast POS / Square POS 모두 중립 색상 기반
```

### 던킨 오렌지 (#FF671F) 허용 사용처
- 헤더 매장명 옆 브랜드 뱃지
- 마감 임박 배너 강조 텍스트
- 발주 완료 상태 체크 아이콘
- 긴급 신선도 경고 포인트

### 던킨 오렌지 금지 사용처
- CTA 버튼 배경 (블랙 유지)
- 탭 active 색상
- 일반 액센트 반복 사용

## 4. 그림자 시스템 (신규)

```css
/* 카드 기본 elevation */

/* 카드 호버·포커스 elevation */

/* 바텀시트 */
--shadow-sheet: 0 -4px 24px rgba(0,0,0,0.12), 0 -1px 4px rgba(0,0,0,0.06);
```

**카드에서 border 제거 → shadow-card-sm으로 대체**

## 6. 컴포넌트 변경 명세

### Card
```
현재: bg-card rounded-2xl p-4 (border 없음 or ring)
변경: bg-card rounded-2xl p-4 shadow-card-sm
      hover 시 shadow-card-md 전환 (transition 200ms)
```

### Button (lg — 주요 CTA)
```
현재: rounded-xl h-14
변경: rounded-full h-14   ← Skip CTA 스타일
      (sm/md는 rounded-xl 유지)
```

### BottomNav
```
현재: bg-card border-t border-border
변경: bg-white/90 backdrop-blur-xl shadow-sheet border-t-0
      active indicator: 아이콘 아래 2px 검정 dot
```

### 모바일 헤더 (layout.tsx)
```
현재: 단순 검정 배경 + 매장명/날씨/시각 작게
변경: 매장명 크게 (text-xl font-bold)
      날씨·시각 우측 정렬 + 아이콘 추가
      하단 라운드 처리 (rounded-b-3xl)
      → Flighty 다크 히어로 느낌
```

## 8. 예측 UI Glow 시스템

### 개념
행동 패턴 학습 → 다음 액션 예측 → 버튼에 ambient glow 표시

### 파일 구조
```
src/shared/lib/actionPredictor.ts   - 순수 함수 (localStorage 기반)
src/shared/model/useActionPredictor.ts - React Hook
```

### 알고리즘
1. **시간대 기본값** — 히스토리 없어도 동작
   - 오전 6-10시: 발주 승인 glow
   - 점심 11-14시: POS 결제 glow
   - 저녁 18-22시: 마감 승인 glow
2. **시퀀스 패턴** (가중치 50%) — A 다음엔 B
3. **시간대 빈도** (가중치 30%)
4. **전체 빈도** (가중치 20%)

### Glow 레벨
```css
glow-low  (확신도 25-45%): 정적 subtle shadow
glow-mid  (확신도 45-75%): 2.5초 pulse 애니메이션
glow-high (확신도 75%+):   1.8초 강한 pulse
```

### 적용 대상
| 버튼 | Action ID |
|------|-----------|
| 하단 탭 (4개) | `nav-home` `nav-order` `nav-assistant` `nav-review` |
| 홈 발주 승인 CTA | `home-order-cta` |
| AI발주 일괄 승인 | `order-approve-all` |
| AI비서 퀵버튼 3개 | `assistant-quick-*` |
| POS 카드결제 | `pos-payment-card` |
| POS 간편결제 | `pos-payment-easy` |
